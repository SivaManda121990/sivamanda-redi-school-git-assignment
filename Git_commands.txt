git clone <repo-url>
  - Clones (copies) a remote Git repository to your local machine.

git status
  - Shows the current state of the working directory and staging area.

git add <file>
  - Adds changes in the specified file to the staging area.

git commit -m "message"
  - Records the staged changes in the repository with a descriptive message.

git push origin main
  - Uploads local commits to the remote repository's main branch.

git log --oneline
  - Shows a concise list of commits with their messages.

git reset HEAD <file>
  - Unstages a file from the staging area.

These are Git commands with explanation.
